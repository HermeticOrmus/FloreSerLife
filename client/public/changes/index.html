Security Hardening

  • Added Helmet.js with Content Security Policy, frame options, and permissions policy
  • Added rate limiting: 500 req/15min global, 5 req/15min for auth endpoints
  • Added security headers in vercel.json: X-Frame-Options DENY, X-Content-Type-Options nosniff, strict Referrer-Policy
  • Added immutable caching for Vite-hashed assets (/assets/*)
  • Removed dead code: server/replitAuth.ts and archived landing page
  • Guarded dev-only route (/design-system-test) behind import.meta.env.DEV
  • Removed console.log statements from production code paths
9

Accessibility Improvements

  • Added skip-to-content link in App.tsx with id="main-content" target
  • Added SheetTitle for mobile navigation accessibility (required by Radix dialog)
  • Added aria-label attributes on search inputs and password toggle buttons
  • Associated htmlFor/id on admin dashboard form labels
  • Wrapped decorative emojis in <span aria-hidden="true">
  • Added width and height attributes on logo images to prevent CLS
  • Added loading="lazy" on below-fold images and fetchPriority="high" on hero images
