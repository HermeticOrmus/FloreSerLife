## Generation Tips

1. **Consistency**: Generate in batches by category
2. **Size**: Always specify size in prompt for proper scale
3. **Testing**: Test each icon at actual use size before finalizing
4. **Variants**: Create hover/active states for interactive elements
5. **Retina**: Generate @2x for crisp display on high-DPI screens
