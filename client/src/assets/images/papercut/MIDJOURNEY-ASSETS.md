## Post-Processing Notes

### For Textures (1.x, 4.x)
- Verify seamless tiling in Photoshop/GIMP
- Export as PNG with compression
- Test at 200% zoom for web quality

### For Hero Elements (2.x)
- Remove backgrounds if not transparent
- Layer in Figma/Photoshop to compose final sprout
- Export composite and individual pieces

### For Pollinators (3.x)
- Ensure transparent backgrounds
- Resize to consistent 48-52px for web display
- Create @2x versions for retina (96-104px)

### For Patterns (4.x)
- Test overlay at 5%, 10%, 15% opacity
- Verify no harsh edges when tiled
